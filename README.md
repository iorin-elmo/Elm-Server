# Elm-Server  
elm-server  
simple chat system  

server system by [Yuya Aizawa](https://github.com/YuyaAizawa/ElmServer)  

## Build  
``` sh  
elm make Client.elm --output=client.js  
elm make Server.elm --output=Server.js  
```  
## Run  
node version 10 or above required  
``` sh  
node start-server.js  
```
